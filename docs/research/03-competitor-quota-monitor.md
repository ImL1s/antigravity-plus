# 競品分析：Antigravity Quota Monitor

## 基本資訊

| 項目 | 說明 |
|------|------|
| **名稱** | Antigravity Quota Monitor |
| **平台** | VS Code / Open VSX |
| **類型** | 配額監控擴展 |
| **用途** | 監控 Antigravity AI 配額、積分和使用限制 |

---

## 核心功能

### 1. 即時狀態列追蹤器 📊
- 直接在 IDE 狀態列顯示配額狀態
- 一目了然的使用率顯示

### 2. 詳細 AI Dashboard 🎛️
顯示的資訊包括：
- 各模型特定使用量
- Token 限制
- 積分刷新時間

### 3. 智慧使用警報 ⚠️
- 顏色編碼指示器
- 配額接近耗盡時警告
- 不同警報等級：
  - 🟢 充足
  - 🟡 中等
  - 🔴 低/耗盡

---

## UI 設計參考

### 狀態列顯示格式
```
[📊 45.2K / 100K tokens | ⏱️ 刷新: 2h 30m]
```

### Dashboard 面板
可能包含的區塊：
1. **配額概覽卡片** - 當前使用率 / 總量
2. **模型分類統計** - Gemini Pro, Gemini Flash 等
3. **時間軸圖表** - 使用量趨勢
4. **刷新倒計時** - 距離下次配額刷新

## 技術實作推測

### 配額資訊獲取
可能的方式：
1. 解析 Antigravity 日誌文件
2. 攔截 API 回應中的配額資訊
3. 調用 Antigravity 內部 API（如果有公開）

### 本地儲存
需要儲存的資料：
- 歷史使用記錄
- 用戶偏好設定
- 警報閾值配置

使用 VS Code 的 `ExtensionContext.globalState` 或 `workspaceState`
