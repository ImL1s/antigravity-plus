---

## 核心功能

### 1. 自動接受文件編輯 ✅
- 無需手動點擊確認 AI 建議的檔案修改
- 檔案變更自動被接受

### 2. 自動執行終端指令 ✅
- 自動允許 Agent 執行 shell 指令
- 無需每次都手動確認

### 3. 恢復卡住的 Agent 🔄
- 偵測無回應或卡住的 Agent
- 自動恢復執行

### 4. 安全黑名單 🛡️
- 阻擋潛在危險指令
- 預設阻擋 `rm -rf /` 等破壞性指令

### 5. Impact Dashboard 📊
追蹤生產力提升指標：
- 節省的點擊次數
- 節省的時間
- 活躍的 AI 會話數
- 已阻擋的危險指令數

---

## 安裝方式

```
1. 在 Antigravity 或 VS Code 中按 Ctrl+Shift+X 開啟 Extensions
2. 搜索 "auto-all-Antigravity"
3. 點擊 Install
```

---

## 技術實作推測

### 自動核准機制
可能的實作方式：
1. 監聽 VS Code 的對話框事件
2. 攔截 Agent 的核准請求
3. 自動觸發「接受」動作

### Agent 恢復機制
可能的實作方式：
1. 監控 Agent 活動狀態
2. 設定 timeout 閾值
3. 偵測到卡住後觸發恢復操作

### 黑名單實作
可能的實作方式：
1. 攔截終端指令執行請求
2. 對比指令與黑名單規則（支援正則表達式）
3. 符合則阻擋並通知用戶
