# Auto-Accept Workflow

## Trigger
- When the user wants to enable auto-acceptance of Antigravity prompts.

## Steps
1. User installs the extension.
2. User ensures Python and dependencies (`pyautogui`, `opencv-python` if needed) are installed.
3. User captures `accept_button.png` of the prompt to be clicked.
4. User runs command `Antigravity: Start Auto-Accept`.
